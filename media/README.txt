這個資料夾用於存放每日靈修的圖片和音頻檔案。

This folder is for storing daily meditation images and audio files.

檔案命名規則 File Naming Convention:
- 圖片 Image: WhatsApp Image YYYY-MM-DD.jpg
- 音頻 Audio: WhatsApp Audio YYYY-MM-DD.mp3

範例 Examples:
- WhatsApp Image 2026-01-31.jpg
- WhatsApp Audio 2026-01-31.mp3
- WhatsApp Image 2026-02-01.jpg
- WhatsApp Audio 2026-02-01.mp3
